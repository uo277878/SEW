Lenguaje Rust

Ventajas y desventajas

Al ser un lenguaje tan nuevo, realmente se pueden encontrar pocas ventajas y desventajas sobre él.

Ventajas

  • Rust ofrece soporte para diferentes funciones complejas, por lo que puede usarlo para lograr muchas cosas.​
    • Ofreciendo un amplio abanico de posibilidades.
  • Realizar la depuración y las pruebas con Rust es mucho más fácil en comparación con otras plataformas.
    • Teniendo en cuenta lo importantes que son las pruebas y la depuración en la programación, es una de sus mayores ventajas.
  • Puede escribir código que funcione en diferentes plataformas, como Windows y Mac.
    • No cierra puertas a ningún tipo de sistema operativo.

Desventajas

  • Puede pasar mucho tiempo tratando de aprender Rust debido a lo complejo que es.​
    • Es recomendable tener un conocimiento previo de C o C++.
  • Mayor tiempo de compilación y el código compilado puede ser ineficaz.
    • Puede hacernos perder tiempo.
  • Debido a las complejidades, el mantenimiento de Rust puede ser una tarea ardua.
    • Por lo que existe la posibilidad de que se opte por otro lenguaje de programación más sencillo.
